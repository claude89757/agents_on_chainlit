# agents_on_chainlit

基于chainlit和autogen的多智能体应用


## 整体架构

```markdown

          ┌────────────────────────────────────────────────────────────┐
WebUI     │                     Chainlit 交互层                         │
          └────────────────────────────────────────────────────────────┘
          ─────────────────────────────────────────────────────────────
          ┌────────────────────────────────────────────────────────────┐
Agent     │                     Autogen 智能体                          │
          └────────────────────────────────────────────────────────────┘
          ─────────────────────────────────────────────────────────────
          ┌────────────────────────────────────────────────────────────┐
WebUI     │                     Openrouter 大模型服务                    │
          └────────────────────────────────────────────────────────────┘
          ─────────────────────────────────────────────────────────────
          ┌────────────────┐    ┌────────────────┐    ┌────────────────┐  
Resource  │ Mcp Tools      │    │ Function Tools │    │    Others      │   
          └────────────────┘    └────────────────┘    └────────────────┘
```

## 目录结构

```bash
├── README.md               # 项目说明文档
├── chainlit.md             # Chainlit配置说明
├── requirements.txt        # 项目依赖文件
├── .env.example            # 环境变量示例文件
├── public/                 # 公共资源目录
├── src/                    # 源代码目录
    ├── configs/            # 配置文件目录
    ├── app.py             # 主应用程序入口
    ├── auth/               # 认证相关模块
    ├── chat_profiles/      # 聊天配置文件
    ├── chat_handlers/      # 聊天处理模块
    ├── mcp_handler/        # MCP处理模块
    └── agents/             # 智能体模块
```

## 安装依赖

```bash
pip install -r requirements.txt
```

## 环境变量配置

```bash
cd agent_on_chainlit
cp .env.example .env
vim .env  # 添加对应的环境变量信息
```

## 启动方式

```bash
chainlit run src/netgen_app.py  -w 
```

## 访问方式

```bash
浏览器访问: http://localhost:8000
```
